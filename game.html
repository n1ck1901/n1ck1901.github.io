Fluxbox Sauce How To

dnf install gcc-c++
dnf install libX11-devel
Download and extract tar
Go into extracted folder
su -
./configure
make
make install

gedit /usr/share/xsessions/fluxbox.desktop

In this file add

[Desktop Entry]
Name=ANYTHING
Comment=This is fluxbox
Exec=/usr/local/bin/startfluxbox

Now full restart
