Fluxbox Sauce How To

dnf install gcc-c++
dnf install libX11-devel
wget http://sourceforge.net/projects/fluxbox/files/fluxbox/1.3.7/fluxbox-1.3.7.tar.gz
cd /home/USERNAME/Downloads
tar -zxvf fluxbox-1.3.7.tar.gz
cd fluxbox-1.3.7
su -
./configure
make
make install

gedit /usr/share/xsessions/fluxbox.desktop

In this file add

[Desktop Entry]
Name=ANYTHING
Comment=This is fluxbox
Exec=/usr/local/bin/startfluxbox

Now full restart
